# Rongsheng Zhang - 个人学术网站

这是一个为研究生申请而设计的个人学术网站，展示我在数学、统计学、机器学习和数据科学领域的研究成果和学术背景。

## 🌟 网站特色

- **现代化设计**: 响应式布局，支持各种设备
- **学术专业性**: 突出研究经历和学术成就
- **交互体验**: 流畅的动画效果和用户交互
- **内容完整**: 包含教育背景、研究经历、博客和联系方式

## 📁 文件结构

```
dream/
├── index.html          # 主页
├── about.html          # 关于我页面
├── research.html       # 研究经历页面
├── blog.html          # 博客页面
├── contact.html       # 联系页面
├── styles.css         # 样式文件
├── script.js          # 交互脚本
├── package.json       # 项目配置
└── README.md          # 说明文档
```

## 🚀 快速开始

### 方法一：直接打开
1. 下载所有文件到本地
2. 双击 `index.html` 在浏览器中打开

### 方法二：使用本地服务器
```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 或者使用简单HTTP服务器
npm start
```

## 🎨 设计特点

- **色彩方案**: 蓝紫色渐变主题，体现学术专业性
- **字体**: Inter字体，现代简洁
- **图标**: Font Awesome图标库

- **Google Fonts**: 字体库

## 📝 自定义指南

### 修改个人信息
1. 编辑各个HTML文件中的个人信息
2. 更新联系方式
3. 修改研究经历和项目描述

### 添加博客文章
1. 在 `blog.html` 中添加新的文章卡片
2. 设置正确的 `data-category` 属性
3. 更新博客统计数字

### 修改样式
1. 编辑 `styles.css` 文件
2. 主要颜色变量在文件顶部
3. 响应式断点在文件底部

## 🌐 部署到GitHub Pages

1. 将所有文件上传到GitHub仓库
2. 仓库名必须是 `用户名.github.io`
3. 在仓库设置中启用GitHub Pages
4. 选择主分支作为源
5. 网站将在 `https://用户名.github.io` 访问
